## Projects

### 💳 Credit Risk ML System &nbsp; · &nbsp; [Repository →](https://github.com/SOUMYAJYOTI1234/credit_risk_ml_system)

A production-grade, end-to-end ML system for credit card default prediction using the UCI dataset (30,000 records). Built to reflect real industry standards — not just model training, but the full lifecycle.

| Component | Detail |
|---|---|
| **Modelling** | Logistic Regression · Random Forest · XGBoost; selected by cross-validated ROC-AUC |
| **Feature Engineering** | Credit utilisation ratio · payment delay score · average billing amount |
| **Tuning** | Optuna hyperparameter optimisation |
| **Explainability** | SHAP global & local feature attributions |
| **Serving** | FastAPI REST API with calibrated probabilities & Swagger UI |
| **Infrastructure** | Docker · GitHub Actions CI/CD (lint + pytest) |
| **Monitoring** | KS-test & PSI drift detection with automated retraining alerts |

`Python` &nbsp; `XGBoost` &nbsp; `FastAPI` &nbsp; `Docker` &nbsp; `SHAP` &nbsp; `Optuna` &nbsp; `GitHub Actions`
